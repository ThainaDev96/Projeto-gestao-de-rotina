# Projeto-gestao-de-rotina
projeto desenvolvido para ajudar na organização do dia a dia, permitindo que o usuário planeje, categorize e acompanhe suas atividades. Ele conta com notificação, auxiliando o usuário a não esquecer tarefas, categorias, que permitem classificar tarefas e rotinas por tipo, tarefas,com cadastro, edição e controle de tarefas
