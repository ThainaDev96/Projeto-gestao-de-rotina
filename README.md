# Projeto-gestao-de-rotina
Projeto desenvolvido para ajudar na organização do dia a dia, permitindo que o usuário planeje, categorize e acompanhe suas atividades. Ele conta com os seguintes arquivos principais:
-Notificação: auxiliando o usuário a não esquecer tarefas. 
-Categoria: que permitem classificar tarefas e rotinas por tipo.
-Tarefas: com cadastro, edição e controle de tarefas.
-Rotina: responsável por gerenciar atividades recorrentes, incentivando constância, disciplina e melhor aproveitamento do tempo.
